[ ] (Feature 3) Core: Implement Salience Assessment

Create a new AiClient function assess_importance(text).

This function will evaluate any new information (from news, chats, or hub mentions) and return a structured response (e.g., {"importance_score": 1-10, "reason": "...", "tags": [...]}).

Create a new periodic "reflection" trigger (e.g., every few hours).

During this trigger, the agent scans its recent important memories (using the score from Feature 3).

[ ] (Feature 8) Feat (Proactive User Interaction)

Use the "reflection" module (Feature 5) to generate insights about its relationship with the user.

The agent should occasionally create a new local mention (not on the hub) to share these thoughts or ask a question directly to the user/creator.
